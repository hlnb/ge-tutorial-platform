# Black Swan Bistro - HTML Basics Reference

This is the HTML foundation version of the Black Swan Bistro project.

It supports **Part 1** of the learning flow, where students build the homepage structure first before styling it and later expanding the project into a fuller multi-page site.

## How This Reference Fits the Project

- **Part 1** focuses on semantic HTML and homepage structure
- **Part 2** adds layout and styling with CSS
- **Part 3** introduces reusable sections and repeated patterns
- **Part 4** grows the project into a more complete multi-page site

So this HTML reference should be understood as the **starting structure**, not the final finished brief implementation.

## What Students Should Build at This Stage

The homepage foundation should include:

- a clear header and navigation
- a hero / welcome section
- an about section
- a featured menu section
- practical details such as hours and contact information

At this stage, it is completely fine for navigation links to point to sections on the same page.

## How It Lines Up With the Project Brief

The project brief shows the broader direction of the whole Black Swan Bistro build. This HTML reference lines up with it by establishing the core homepage content and shared site language:

- **Home**
- **Menu**
- **About**
- **Contact**

Those labels matter because they become the repeated navigation structure used later when students build additional pages.

## What Is Intentionally Simpler Here

This HTML stage does **not** need to include every higher-fidelity wireframe detail yet.

Students are still learning how to:

- organise content with semantic elements
- build logical section structure
- write readable headings and supporting copy
- prepare a page that CSS can style later

That means the HTML reference is simpler than the later project brief wireframes on purpose.

## What Students Should Notice

### Semantic structure

Look for meaningful HTML elements such as:

- `header`
- `nav`
- `main`
- `section`
- `footer`

These make the page easier to understand, style, and expand later.

### Repeated page ideas

Even in a single-page beginner version, students can start noticing patterns that will matter later:

- consistent headings
- grouped content blocks
- navigational labels
- clear calls to action

These become the basis for reusable components in later parts.

## How To Use This Reference

Students can use this page to:

- compare their own HTML structure
- check whether their content sections are in a sensible order
- confirm they are using semantic elements
- prepare for the CSS and layout work that comes next

## Next Step

Once the HTML structure is in place, students should move to the CSS stage and use the project brief to guide:

- colour choices
- spacing
- hierarchy
- layout decisions
- consistency with the Black Swan Bistro wireframes
